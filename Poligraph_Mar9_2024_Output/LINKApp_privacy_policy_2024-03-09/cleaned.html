II. MOBILE HELIX LINK APP AND SYSTEM, “APPLICATION,” PRIVACY POLICY

This privacy policy governs the use of the software application, Mobile Helix Link App and System (“Application”), that was created by Mobile Helix, Inc. (“we” or “us”). This privacy policy explains what and how we collect, use, and disseminate information obtained from or about the employee of the hosting entity as a user of the Application.

For background, the Application is licensed and managed by hosting entities, such as law firms. The hosting entity installs the Application on servers which are managed by the entity. Mobile Helix does not have access to those servers. The hosting entity distributes the Application to its employees. The Application connects with the hosting entity’s enterprise resources and enables users/employees to access the data, files, and applications which they use for work. The Application is encrypted, remotely wipeable, authenticated using certificates as well as other methods including Single Sign-on. The Application is not licensed to or accessible for use by consumers.

Mobile Helix does not directly interface with the hosting entity’s employees nor with the hosting entity’s clients. The Application does not report any information about the users of the Application to Mobile Helix. An entity may provide limited information via email or via a support request. Support requests are managed by Mobile Helix in a top-tier, security vetted issue management system.

Types of Information Collected

User Provided Information

When the employee is provisioned to the Application by their employer, they are registered by the employer with their name, email address, user name, password and other registration information. This information is already known to, stored by, and managed by their employer (entity) which is hosting the Application, e.g., the law firm.

Mobile Helix does have access to, nor collect, nor store this information.

Automatically Collected Information

  1. The Application may employ the employee’s location to find a network gateway for optimal global performance, but the Application does not track the user’s location. The employee’s location is never stored (not even in the Application’s analytics). The hosting entity, e.g., a law firm, may disable the use of the user’s location to find a network gateway.
  2. Mobile Helix does not have any access in any way to the employee’s location or to the employee’s activity in the Application. The user’s actions inside of the Application are confidential to the hosting entity, e.g., a law firm.
  3. The Application does collect metadata (analytics) about the user’s activities in the Application so that the hosting entity may use that metadata to determine how the Application is being used. This metadata only includes the applications that the employee accesses in the Application and the activities that the employee executes (e.g., to compose an email or to open a file) in the Application. The Application never tracks, stores, processes, monitors, etc., any of the actual data contained in, for example, an email or a file, that the employee views in the Application. In any case, this metadata is not accessible to Mobile Helix unless the entity grants one-time access to this data.

When the employee interacts with the Application, the Application may collect certain information automatically, including, but not limited to, the type of device used, the device’s unique device ID, the IP address of the device, and the operating system. As in point number two above, Mobile Helix does not have access in any way to this data. This data is confidential to the hosting organization or firm.

Use of Information

Mobile Helix cannot access User Provided Information and Automatically Collected Information. It is only accessible by the hosting entity, e.g., a law firm.

Mobile Helix does not use such information.

Sharing of Information

Mobile Helix cannot access User Provided Information and Automatically Collected Information. It is only accessible by the hosting entity, e.g., a law firm.

Mobile Helix does not share such information.
