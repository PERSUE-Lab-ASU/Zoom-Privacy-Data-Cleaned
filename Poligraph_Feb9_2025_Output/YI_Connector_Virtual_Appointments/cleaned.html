9. Zoom Customer Data Retention

The following Zoom customer data is stored in our system, only while the Connector is connected to Zoom and authorized:

  • Zoom Account ID
  • Zoom User ID
  • Zoom API Access Token
  • Zoom API Refresh Token

This data is deleted from our system when a user chooses to disconnect from Zoom through their Connector account, or if they deauthorize the Connector through their Zoom account. When deauthorizing the Connector from their Zoom account, users may select a data retention preference with the option to allow the Connector to retain their user information. In all cases we delete the user data from our system, regardless of this preference.

10. Google Customer Data Retention

The following Google customer data is stored in our system, only while the Connector is connected to Google and authorized:

  • Google API Access Token
  • Google API Refresh Token

This data is only used for the following purposes:

  • To retreive and display the name of the user's Primary Google Calendar only, for the sole purpose of verifying the user has successfully authorized the Connector app to read/write data on the user's primary Google calendar.
  • To create calendar events on the user's primary Google calendar.
  • To delete only calendar events that the Connector has first created on the user's primary Google calendar

This data is deleted from our system when a user chooses to disconnect from Google through their Connector account. When a user disconnects from Google through their Connector account, the Connector app is also deauthorized in the user's Google account.
