The User may give his consent to the use of the Cookies described in section 12 above using the methods indicated on the banner published on the Platform

The manifestation of consent can be modified at any time by the User according to the indications contained in this Information, which is accessible via the link in the banner and on each page of the Platform (footer). To this end, the User can also contact WeSchool at any time to obtain information about the concrete ways in which he can exercise his consent / selective refusal or delete cookies.

However, WeSchool specifies that disabling the technical cookies indicated in the previous section 12 could prevent correct navigation on the Platform and / or limit its usability. Consequently, in order to better use the WeSchool Services accessible through the Platform, the User is strongly discouraged from proceeding with the deactivation of these technical cookies.

Having said this, with reference to all the Cookies of the Platform, the User can change the settings relating to them, in the manner provided for in the control panels of the browser used, such as by way of example:

  • Google-Chrome: https://support.google.com/chrome/answer/95647?hl=it-IT&hlrm=fr&hlrm=en
  • Mozilla-Firefox: https://support.mozilla.org/it/kb/Attivare%20e%20disattivare%20i%20cookie
  • Internet Explorer: https://support.microsoft.com/it-it/help/278835/how-to-delete-cookie-files-in-internet-explorer
  • Microsoft Edge: https://support.microsoft.com/it-it/help/4027947/microsoft-edge-delete-cookies
  • Safari: https://support.apple.com/it-it/guide/safari/sfri11471/mac

Furthermore, with specific reference to profiling and advertising cookies, in addition to being able to use the tools provided by the browser to activate or deactivate individual cookies, WeSchool informs the User that the website www.youronlinechoices.com shows the list of the main providers that they work with website managers to collect and use information useful for the use of behavioral advertising. The User can deactivate or activate all companies (including WeSchool) or, alternatively, adjust their preferences individually for each company. To do so, the User can use the tool on the page www.youronlinechoices.com/it/le-tue-scelte to check any preference on behavioral advertising provided when browsing the Internet.
